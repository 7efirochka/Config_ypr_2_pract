Визуализатор зависимостей пакетов - инструмент для анализа и отображения графов зависимостей npm-пакетов. Поддерживает построение полных цепочек зависимостей, обнаружение циклов и визуализацию в форматах Mermaid и ASCII. Работает как с реальными пакетами через NPM Registry, так и с тестовыми данными.
